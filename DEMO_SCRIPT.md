# Demo Video Script

Use this for a short buyer-facing demo.

Target length: 2 to 3 minutes.

## Opening

Say:

> This is Personal X Operator, a $20 local-first X engagement assistant. It scans your X feed, finds relevant conversations, drafts replies, and can post from your own browser when you enable live mode.

Show:

- the landing page at `/`
- the $20 one-time purchase message
- the "Not a follower bot" boundary

## Dashboard

Say:

> The dashboard is intentionally simple. No login, no SaaS account, no teams, no billing screen. It runs locally.

Show:

- `/dashboard`
- AI mode
- Post mode
- Reply mode
- posts today
- replies today
- feed scanned
- recent posts/replies/logs

## Safe Draft Mode

Say:

> Draft mode is the default. Nothing posts live until you choose live posting or live replies.

Show:

- `Generate post draft`
- `Scan and draft replies`
- `Enable live posting`
- `Enable live replies`

## X Browser Session

Say:

> It uses Playwright with a persistent browser session. You log into X once, and the local agent reuses that browser profile.

Show:

- click `Open X browser`
- X opening in the separate browser window

## Local Curator Mode

Say:

> OpenAI is optional. Local curator mode can score and draft without API credits. OpenAI mode is there if you want richer generation.

Show:

- AI mode as `Local`
- `Use OpenAI` button

## Safety

Say:

> It does not follow or unfollow accounts. It blocks sensitive topics like politics, religion, legal, medical, tragedy, harassment, adult content, and drama. You are responsible for what your X account posts.

Show:

- `forbidden.md`
- dashboard skipped candidates/logs

## Close

Say:

> If you want a simple local tool for finding better X conversations without living in the feed, this is the whole thing: $20 once, run it locally, keep your data on your machine.

End on:

- landing page
- purchase button
